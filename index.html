Consent2Share Project

Consent2Share is an open source software application sponsored by the U.S. Substance Abuse and Mental Health Administration (SAMHSA) which is designed to support Behavioral Health's integration with Health Information Exchanges (HIE).

The source code is available here: https://github.com/BHITS/Consent2Share.
